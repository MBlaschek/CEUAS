===========
Radiosonde Tools
===========

Radiosonde Tools a(rasotools) re a collection of methods for calculating
anomalies and other diagnostics from multilevel radiosonde
time series. Typical usage
often looks like this::

    #!/usr/bin/env python

    from rasotools import location
    from rasotools import anomaly

    x=anomaly(y)

(Note the double-colon and 4-space indent formatting above.)

Paragraphs are separated by blank lines. *Italics*, **bold**,
and ``monospace`` look like this.
